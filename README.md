Computer Vision Projects
A collection of deep learning models and computer vision architectures implemented from scratch using Python and PyTorch, covering both foundational and state-of-the-art approaches.
Projects & Architectures Implemented

AlexNet – Pioneer deep CNN for image classification
VGGNet – Deep convolutional network with uniform architecture
GoogLeNet – Inception-based architecture with parallel convolutions
ResNet – Residual learning to train very deep networks
DenseNet – Densely connected convolutional networks
MobileNet – Lightweight CNN optimized for efficiency
EfficientNet – Scalable and accurate CNN architecture
SqueezeNet – Compact CNN with AlexNet-level accuracy
NASNet – Neural Architecture Search-based model
ViT (Vision Transformer) – Transformer applied to image patches
Attention Mechanism – Self-attention for visual feature focus
Transfer Learning – Fine-tuning pretrained models on custom datasets
OpenCV – Classical computer vision techniques
