**Computer Vision Projects**
A collection of deep learning models and computer vision architectures implemented from scratch using Python and PyTorch, covering both foundational and state-of-the-art approaches.

**Projects & Architectures Implemented**

1.  AlexNet – Pioneer deep CNN for image classification
2.  VGGNet – Deep convolutional network with uniform architecture
3.  GoogLeNet – Inception-based architecture with parallel convolutions
4.  ResNet – Residual learning to train very deep networks
5.  DenseNet – Densely connected convolutional networks
6.  MobileNet – Lightweight CNN optimized for efficiency
7.  EfficientNet – Scalable and accurate CNN architecture
8.  SqueezeNet – Compact CNN with AlexNet-level accuracy
9.  NASNet – Neural Architecture Search-based model
10. ViT (Vision Transformer) – Transformer applied to image patches
11. Attention Mechanism – Self-attention for visual feature focus
12. Transfer Learning – Fine-tuning pretrained models on custom datasets
13. OpenCV – Classical computer vision techniques
